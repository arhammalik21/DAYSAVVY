About DAYSAVVY

DAYSAVVY is your intelligent task management assistant, designed to help you organize and optimize your day with ease.

Key Features

  • Task creation, categorization, editing, and deletion
  • Date and time tracking for all tasks
  • AI voice integration for hands-free task management
  • Responsive and user-friendly interface
  • Secure backend database management

Technologies Used

  • Flask & Python
  • SQLAlchemy
  • HTML, CSS, JavaScript
  • AI Voice Command Integration
